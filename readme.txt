Customize frontend messages returned by Awesome Support.

== Description ==

Awesome Support - Messages is a very easy-to-use plugin for Awesome Support.

It adds the "Messages" tab in your admin Tickets -> Settings screen where you can override frontend messages with the text you want. Turn the "You are not allowed to submit a ticket." message into "Only contributors can submit a ticket." or anything you want.

There's a [GIT repository](https://github.com/rubengc/awesome-support-messages) too if you want to contribute a patch.

This plugin requires (Awesome Support)[https://es.wordpress.org/plugins/awesome-support/].

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
